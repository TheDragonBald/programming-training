# Заявки — система учёта обращений

ТЗ: Веб-приложение на Django для приёма и просмотра клиентских заявок.

## Стек

- Python 3.х (Тут 3.14.4)
- Django
- Bootstrap 5 (через CDN)

## Установка и запуск

1. Клонировать репозиторий и перейти в папку проекта:
   `git clone git clone https://github.com/TheDragonBald/programming-training.git` → `cd programming-training/test_tasks/Nikolsky_help_test`
2. Создать и активировать виртуальное окружение:
   `python -m venv venv` → `.\venv\Scripts\activate`
3. Установить зависимости: `pip install -r requirements.txt`
4. Применить миграции: `python manage.py migrate`
5. Создать суперпользователя: `python manage.py createsuperuser`
6. Запустить сервер: `python manage.py runserver`

## Страницы

- `/requests/` — список заявок (таблица, поиск, цветные статусы)
- `/requests/create/` — форма создания заявки
- `/admin/` — админ-панель

## Функционал

- Модель заявки: имя, телефон, email, тема, текст, статус, дата создания
- Админ-панель: список, поиск по имени/телефону/email, фильтр по статусу, дата создания только для чтения
- Список заявок с Bootstrap-таблицей, цветными бейджами статусов и живым поиском по таблице (JS на стороне браузера)
- Форма создания с валидацией, сообщением об успехе и редиректом

## SEO-элементы

- `<title>` — уникальный заголовок страницы (вкладка браузера и поисковая выдача)
- `<meta name="description">` — описание-сниппет для поисковой выдачи
- `<h1>` — один осмысленный главный заголовок на каждой странице
- Корректная структура документа: `lang="ru"`, кодировка UTF-8, `viewport` для адаптивности
